Site Alten

Objectif du projet

L’objectif de ce projet était de coder un site internet pour des élèves de 3e à la recherche d’un stage dans une entreprise. Notre but était de rendre accessibles les informations présentes sur les sites officiels en les simplifiant dans une interface plus intuitive, adaptée à leur âge et leurs habitudes numériques.

Technologies utilisées

  • HTML
  • CSS
  • JavaScript

Mon travail dans ce projet

Mon travail dans ce projet étais de réfléchir à une maquette du site pour se conformer au attente qui nous avaient été donnée je me suis également occuper de la création de la page "Le Groupe" qui fais partie du site internet, je me suis occuper de la partie HTML, CSS et JavaScript
